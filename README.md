# 🌾 Rice Plant Disease Detection System

## 📌 Overview
This project focuses on developing an AI-powered system to **detect and classify rice plant diseases** using leaf images. The system leverages **Convolutional Neural Networks (CNN)** to provide accurate disease identification along with **treatment recommendations, fertilizer suggestions, and weather-based disease prediction**.

The goal is to assist farmers in **early disease detection**, reduce crop loss, and promote **sustainable agriculture practices**.

---

## ❗ Problem Statement
Traditional rice disease detection relies on manual observation, which is:
- Time-consuming  
- Prone to human error  
- Not easily accessible to all farmers  

This project aims to automate disease detection and provide **intelligent decision support** using deep learning.

---

## 🚀 Proposed System
The system uses a **CNN-based deep learning model** trained on rice leaf images to:

- Detect and classify diseases (bacterial, fungal, viral)
- Provide **accurate and fast diagnosis**
- Reduce dependency on manual inspection

---

## 🔍 Key Features

### 🧠 Disease Detection
- Upload rice leaf image  
- CNN model identifies disease type with high accuracy  

### 💊 Medicine Recommendation
- Suggests appropriate pesticides/fungicides  
- Helps farmers take **correct treatment actions**  

### 🌱 Fertilizer Recommendation
- Detects nutrient deficiencies  
- Recommends suitable fertilizers  

### 🌦 Weather-Based Prediction
- Uses environmental factors like:
  - Temperature  
  - Humidity  
  - Rainfall  
- Predicts possible disease outbreaks  

### 📱 Mobile & Cloud Access
- Mobile app using **Flutter**  
- Backend using **Flask**  
- Real-time results with user-friendly interface
